##     GradeBook

# 1. 🎯 Presentation du projet

  GradeBook est une appication web de gestion de notes pour un eatblissement secondaire (College / lycee).
  Elle permetsd gerer numeriquements les nots de eleves, calculer les myenne, generer les bulltaeins et affiche les classement par classe.

# 2. 🏫 Niveaux scolaires secondaire

  i. Niveaux scolaire :
    👉🏾 Premier cycle : 6eme; 5eme; 4eme; 3eme;
    👉🏾 Second cycle : seconde (2nde), premiere (1ere), terminale (Tle)
    
  ii. Specificites du secondaire
    ✅ Professseure rincipale par classe
    ✅ 3 trimestre
    ✅ Notes sur 20 avecmoyenne trimestrielles
    ✅ Appreciation du professuer sur le bulletin
    ✅ Absences sur le bulletin
    ✅ Redoublement / passage en classe superieur
